Hello everyone, this is my first Python project, which I wrote after learning the basic functions of the language and the pygame module.

What is this project:
It's a simple 2D game with very basic effects and movement. The game has an initial main menu, a level menu, and six simple levels. I changed the frame rate for two of them, which in turn creates a slow motion effect. I also added rock-hit sounds and gunshot sounds. Music plays in the background.

This project is suitable for most people who want to understand programming. I've described every action in detail in the code, so it shouldn't be difficult to understand. Have a nice day, and thank you for your time.
